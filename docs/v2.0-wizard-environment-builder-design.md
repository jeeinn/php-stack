## 📋 项目概述

### 背景
当前"软件管理"模块采用逐个安装的方式，用户需要：
1. 分别安装 PHP、MySQL、Redis 等服务
2. 手动处理依赖关系和端口冲突
3. 多次等待容器启动

这种"组装零件"式的操作繁琐且容易出错，不符合开发者"快速搭建开发环境"的核心需求。

### V1.0 目标
将"软件管理"重构为**向导式一键搭建**模式：
- ✅ **3 步完成**：选择技术栈 → 预览配置 → 一键启动
- ✅ **智能配置**：自动处理依赖、端口、网络、数据卷
- ✅ **版本兼容**：内置推荐组合，避免版本冲突
- ✅ **开箱即用**：启动后立即可用于开发
- ✅ **极简设计**：仅支持 Nginx + PHP + MySQL + Redis，移除高级配置

### V1.0 范围界定

#### ✅ 包含功能
- PHP 安装（支持扩展选择，自定义 Dockerfile 构建）
- MySQL 安装（单数据库，固定密码 `root123`）
- Redis 安装（基础配置）
- Nginx 安装（无 SSL，基础虚拟主机配置）
- 镜像源配置（通过 `.env` 文件管理，解决网络问题）
- 向导式 UI（3 步骤）
- 配置预览（YAML 查看）
- 简单日志查看（最近 100 行）

#### ❌ 不包含功能（V2.0+）
- MongoDB 支持
- 多数据库管理
- SQL 文件导入
- SSL 证书配置
- 资源限制配置
- WebSocket 实时日志
- 备份恢复功能
- 环境克隆/快照
- 团队协作功能

### 设计原则
1. **用户优先**：聚焦开发者的真实需求（快速搭建环境）
2. **简化操作**：从"10 步操作"减少到"3 步完成"
3. **智能默认**：提供合理的默认配置，减少用户决策
4. **无需兼容**：项目处于开发阶段，不保留旧方案的冗余代码
5. **渐进增强**：V1.0 最小可用，后续迭代添加高级功能

## 🌐 镜像源配置模块（V1.0 核心功能）

### 设计理念

将镜像源配置从 Dockerfile 中抽离，统一通过 `.env` 文件管理，实现：
- ✅ **集中管理**：所有镜像源配置在一处
- ✅ **灵活切换**：用户可随时修改，无需重新构建镜像
- ✅ **环境隔离**：不同项目可使用不同镜像源
- ✅ **版本控制友好**：`.env.example` 提交到 Git，`.env` 忽略

### 文件结构

```
php-stack/
├── .env                    # 全局配置（不提交到 Git）
├── .env.example            # 配置模板（提交到 Git）
├── src-tauri/
│   └── src/
│       ├── engine/
│       │   ├── mod.rs
│       │   ├── compose_manager.rs
│       │   ├── environment_builder.rs
│       │   └── mirror_config.rs      # 新增：镜像源配置模块
│       └── commands.rs
└── build/
    └── php-custom/
        ├── Dockerfile                # 构建时动态生成
        └── .docker.env               # Docker 构建环境变量（可选）
```

### `.env` 文件格式

```bash
# .env.example（模板文件，提交到 Git）

# ==================== 镜像源配置 ====================

# Docker Hub 镜像加速
# 可选值: default, aliyun, ustc, tencent
DOCKER_REGISTRY_MIRROR=aliyun

# APT 软件源（Debian/Ubuntu）
# 可选值: default, aliyun, tsinghua, ustc
APT_MIRROR=aliyun

# Composer 镜像源
# 可选值: default, aliyun, huaweicloud

# PyPI 镜像源（pip install）

# NPM 镜像源
# 可选值: default, taobao, tencent
NPM_MIRROR=taobao

# ==================== Docker 构建代理 ====================

# HTTP 代理（可选，留空则不使用）
# HTTP_PROXY=http://127.0.0.1:7890
# HTTPS_PROXY=http://127.0.0.1:7890
```

### Rust 实现

```rust
// src-tauri/src/engine/mirror_config.rs

use serde::{Deserialize, Serialize};
use std::collections::HashMap;
use std::fs;
use std::path::Path;

/// 镜像源配置
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct MirrorConfig {
    pub docker_registry: MirrorSource,
    pub apt_mirror: MirrorSource,
    pub composer_mirror: MirrorSource,
    pub pypi_mirror: MirrorSource,
    pub npm_mirror: MirrorSource,
    pub http_proxy: Option<String>,
    pub https_proxy: Option<String>,
    pub no_proxy: Option<String>,
}

/// 镜像源类型
#[derive(Debug, Clone, Serialize, Deserialize, PartialEq)]
pub enum MirrorSource {
    Default,
    Aliyun,
    Tsinghua,
    Ustc,
    Tencent,
    HuaweiCloud,
    Taobao,
}

impl MirrorSource {
    /// 获取镜像 URL
    pub fn get_url(&self, service: &str) -> String {
        match (self, service) {
            // Docker Registry
            (MirrorSource::Aliyun, "docker") => {
                // 注意：阿里云需要用户自己的加速器地址
                "https://<your-id>.mirror.aliyuncs.com".to_string()
            }
            (MirrorSource::Ustc, "docker") => {
                "https://docker.mirrors.ustc.edu.cn".to_string()
            }
            (MirrorSource::Tencent, "docker") => {
                "https://mirror.ccs.tencentyun.com".to_string()
            }
            
            // APT
            (MirrorSource::Aliyun, "apt") => {
                "http://mirrors.aliyun.com/debian/".to_string()
            }
            (MirrorSource::Tsinghua, "apt") => {
                "https://mirrors.tuna.tsinghua.edu.cn/debian/".to_string()
            }
            (MirrorSource::Ustc, "apt") => {
                "https://mirrors.ustc.edu.cn/debian/".to_string()
            }
            
            // Composer
            (MirrorSource::Aliyun, "composer") => {
                "https://mirrors.aliyun.com/composer/".to_string()
            }
            (MirrorSource::HuaweiCloud, "composer") => {
                "https://repo.huaweicloud.com/repository/php/".to_string()
            }
            
            // PyPI
            (MirrorSource::Aliyun, "pypi") => {
                "https://mirrors.aliyun.com/pypi/simple/".to_string()
            }
            (MirrorSource::Tsinghua, "pypi") => {
                "https://pypi.tuna.tsinghua.edu.cn/simple".to_string()
            }
            
            // NPM
            (MirrorSource::Taobao, "npm") => {
                "https://registry.npmmirror.com".to_string()
            }
            (MirrorSource::Tencent, "npm") => {
                "https://mirrors.cloud.tencent.com/npm/".to_string()
            }
            
            _ => String::new(), // Default or unsupported
        }
    }
    
    /// 转换为字符串标识
    pub fn as_str(&self) -> &str {
        match self {
            MirrorSource::Default => "default",
            MirrorSource::Aliyun => "aliyun",
            MirrorSource::Tsinghua => "tsinghua",
            MirrorSource::Ustc => "ustc",
            MirrorSource::Tencent => "tencent",
            MirrorSource::HuaweiCloud => "huaweicloud",
            MirrorSource::Taobao => "taobao",
        }
    }
    
    /// 从字符串解析
    pub fn from_str(s: &str) -> Self {
        match s {
            "aliyun" => MirrorSource::Aliyun,
            "tsinghua" => MirrorSource::Tsinghua,
            "ustc" => MirrorSource::Ustc,
            "tencent" => MirrorSource::Tencent,
            "huaweicloud" => MirrorSource::HuaweiCloud,
            "taobao" => MirrorSource::Taobao,
            _ => MirrorSource::Default,
        }
    }
}

impl MirrorConfig {
    /// 从 .env 文件加载配置
    pub fn load_from_env() -> Result<Self, String> {
        let env_path = Path::new(".env");
        
        if !env_path.exists() {
            log::warn!("⚠️ .env 文件不存在，使用默认配置");
            return Ok(Self::default());
        }
        
        let env_content = fs::read_to_string(env_path)
            .map_err(|e| format!("读取 .env 文件失败: {}", e))?;
        
        let env_map = Self::parse_env_file(&env_content);
        
        Ok(Self {
            docker_registry: MirrorSource::from_str(
                env_map.get("DOCKER_REGISTRY_MIRROR").map(|s| s.as_str()).unwrap_or("default")
            ),
            apt_mirror: MirrorSource::from_str(
                env_map.get("APT_MIRROR").map(|s| s.as_str()).unwrap_or("default")
            ),
            composer_mirror: MirrorSource::from_str(
                env_map.get("COMPOSER_MIRROR").map(|s| s.as_str()).unwrap_or("default")
            ),
            pypi_mirror: MirrorSource::from_str(
                env_map.get("PYPI_MIRROR").map(|s| s.as_str()).unwrap_or("default")
            ),
            npm_mirror: MirrorSource::from_str(
                env_map.get("NPM_MIRROR").map(|s| s.as_str()).unwrap_or("default")
            ),
            http_proxy: env_map.get("HTTP_PROXY").cloned(),
            https_proxy: env_map.get("HTTPS_PROXY").cloned(),
            no_proxy: env_map.get("NO_PROXY").cloned(),
        })
    }
    
    /// 解析 .env 文件内容
    fn parse_env_file(content: &str) -> HashMap<String, String> {
        let mut map = HashMap::new();
        
        for line in content.lines() {
            let line = line.trim();
            
            // 跳过注释和空行
            if line.is_empty() || line.starts_with('#') {
                continue;
            }
            
            // 解析 KEY=VALUE
            if let Some((key, value)) = line.split_once('=') {
                let key = key.trim().to_string();
                let value = value.trim().trim_matches('"').trim_matches('\'').to_string();
                map.insert(key, value);
            }
        }
        
        map
    }
    
    /// 保存到 .env 文件
    pub fn save_to_env(&self) -> Result<(), String> {
        let env_path = Path::new(".env");
        let mut env_content = if env_path.exists() {
            fs::read_to_string(env_path)?
        } else {
            String::new()
        };
        
        // 更新或添加配置项
        env_content = Self::update_env_value(&env_content, "DOCKER_REGISTRY_MIRROR", self.docker_registry.as_str());
        env_content = Self::update_env_value(&env_content, "APT_MIRROR", self.apt_mirror.as_str());
        env_content = Self::update_env_value(&env_content, "COMPOSER_MIRROR", self.composer_mirror.as_str());
        env_content = Self::update_env_value(&env_content, "PYPI_MIRROR", self.pypi_mirror.as_str());
        env_content = Self::update_env_value(&env_content, "NPM_MIRROR", self.npm_mirror.as_str());
        
        if let Some(proxy) = &self.http_proxy {
            env_content = Self::update_env_value(&env_content, "HTTP_PROXY", proxy);
        }
        if let Some(proxy) = &self.https_proxy {
            env_content = Self::update_env_value(&env_content, "HTTPS_PROXY", proxy);
        }
        
        fs::write(env_path, env_content)?;
        log::info!("✅ 镜像源配置已保存到 .env");
        Ok(())
    }
    
    /// 更新 .env 文件中的值
    fn update_env_value(content: &str, key: &str, value: &str) -> String {
        let lines: Vec<&str> = content.lines().collect();
        let mut new_lines = Vec::new();
        let mut found = false;
        
        for line in lines {
            if line.trim().starts_with(&format!("{}=", key)) {
                new_lines.push(&format!("{}={}", key, value));
                found = true;
            } else {
                new_lines.push(line);
            }
        }
        
        if !found {
            new_lines.push(&format!("{}={}", key, value));
        }
        
        new_lines.join("\n")
    }
    
    /// 生成 Docker 构建参数
    pub fn to_build_args(&self) -> Vec<String> {
        let mut args = Vec::new();
        
        // 添加代理配置
        if let Some(proxy) = &self.http_proxy {
            args.push(format!("HTTP_PROXY={}", proxy));
        }
        if let Some(proxy) = &self.https_proxy {
            args.push(format!("HTTPS_PROXY={}", proxy));
        }
        if let Some(no_proxy) = &self.no_proxy {
            args.push(format!("NO_PROXY={}", no_proxy));
        }
        
        args
    }
    
    /// 生成 Dockerfile 中的镜像源配置片段
    pub fn to_dockerfile_snippet(&self) -> String {
        let mut snippet = String::new();
        
        // APT 镜像源
        if self.apt_mirror != MirrorSource::Default {
            let apt_url = self.apt_mirror.get_url("apt");
            snippet.push_str(&format!(
                r#"# 配置 APT 镜像源
RUN sed -i 's|deb.debian.org/debian|{}|g' /etc/apt/sources.list && \
    sed -i 's|security.debian.org|{}|g' /etc/apt/sources.list

"#,
                apt_url.trim_end_matches('/'),
                apt_url.replace("debian", "debian-security").trim_end_matches('/')
            ));
        }
        
        // Composer 镜像源
        if self.composer_mirror != MirrorSource::Default {
            let composer_url = self.composer_mirror.get_url("composer");
            snippet.push_str(&format!(
                r#"# 配置 Composer 镜像源
RUN composer config -g repo.packagist composer {}

"#,
                composer_url.trim_end_matches('/')
            ));
        }
        
        // PyPI 镜像源
        if self.pypi_mirror != MirrorSource::Default {
            let pypi_url = self.pypi_mirror.get_url("pypi");
            snippet.push_str(&format!(
                r#"# 配置 PyPI 镜像源
RUN pip config set global.index-url {}

"#,
                pypi_url.trim_end_matches('/')
            ));
        }
        
        snippet
    }
}
```

```rust
/// 生成自定义 Dockerfile
pub fn generate_dockerfile(
    php_version: &str,
    extensions: &[String],
    mirror_config: &MirrorConfig,
) -> Result<String, String> {
    let mut dockerfile = format!("FROM php:{}-fpm\n\n", php_version);
    
    // 收集所有需要的系统依赖
    let mut all_system_deps = Vec::new();
    let mut core_extensions = Vec::new();
    let mut pecl_extensions = Vec::new();
    
    for ext_name in extensions {
        if let Some(metadata) = ExtensionRegistry::get_extension(ext_name) {
            match metadata.ext_type {
                ExtensionType::Builtin => {
                    // 内置扩展无需安装
                    log::info!("✅ 内置扩展: {}", ext_name);
                }
                ExtensionType::Core => {
                    all_system_deps.extend(metadata.system_deps);
                    core_extensions.push((ext_name.clone(), metadata.config_flags));
                }
                ExtensionType::Pecl => {
                    all_system_deps.extend(metadata.system_deps);
                    pecl_extensions.push((
                        ext_name.clone(),
                        metadata.pecl_version,
                    ));
                }
                ExtensionType::Custom => {
                    return Err(format!("不支持的自定义扩展: {}", ext_name));
                }
            }
        } else {
            return Err(format!("未知扩展: {}", ext_name));
        }
    }
    
    // 配置镜像源（解决网络问题）
    if !mirror_config.is_default() {
        dockerfile.push_str(&generate_mirror_config(mirror_config));
        dockerfile.push('\n');
    }
    
    // 安装系统依赖
    if !all_system_deps.is_empty() {
        all_system_deps.sort();
        all_system_deps.dedup();
        
        dockerfile.push_str("# 安装系统依赖\n");
        dockerfile.push_str("RUN apt-get update && apt-get install -y --no-install-recommends \\\n");
        for (i, dep) in all_system_deps.iter().enumerate() {
            if i < all_system_deps.len() - 1 {
                dockerfile.push_str(&format!("    {} \\\n", dep));
            } else {
                dockerfile.push_str(&format!("    {} \\\n", dep));
            }
        }
        dockerfile.push_str("    && rm -rf /var/lib/apt/lists/*\n\n");
    }
    
    // 安装核心扩展
    if !core_extensions.is_empty() {
        dockerfile.push_str("# 安装 PHP 核心扩展\n");
        for (ext_name, config_flags) in &core_extensions {
            if config_flags.is_empty() {
                dockerfile.push_str(&format!("RUN docker-php-ext-install -j$(nproc) {}\n", ext_name));
            } else {
                dockerfile.push_str(&format!(
                    "RUN docker-php-ext-configure {} {} && \\\n    docker-php-ext-install -j$(nproc) {}\n",
                    ext_name,
                    config_flags.join(" "),
                    ext_name
                ));
            }
        }
        dockerfile.push('\n');
    }
    
    // 安装 PECL 扩展
    if !pecl_extensions.is_empty() {
        dockerfile.push_str("# 安装 PECL 扩展\n");
        for (ext_name, version) in &pecl_extensions {
            if let Some(ver) = version {
                dockerfile.push_str(&format!(
                    "RUN pecl install {}-{} && docker-php-ext-enable {}\n",
                    ext_name, ver, ext_name
                ));
            } else {
                dockerfile.push_str(&format!(
                    "RUN pecl install {} && docker-php-ext-enable {}\n",
                    ext_name, ext_name
                ));
            }
        }
        dockerfile.push('\n');
    }
    
    // 设置工作目录
    dockerfile.push_str("WORKDIR /var/www/html\n");
    
    Ok(dockerfile)
}
```

```rust
/// 构建自定义 PHP 镜像
pub async fn build_custom_php_image(
    php_version: &str,
    extensions: &[String],
    mirror_config: &MirrorConfig,
) -> Result<String, String> {
    // 1. 生成唯一的镜像标签
    let image_tag = generate_image_tag(php_version, extensions);
    
    // 2. 检查镜像是否已存在（缓存命中）
    if image_exists(&image_tag).await {
        log::info!("✅ 使用缓存镜像: {}", image_tag);
        return Ok(image_tag);
    }
    
    // 3. 生成 Dockerfile
    let dockerfile = generate_dockerfile(php_version, extensions, mirror_config)?;
    
    // 4. 创建临时构建目录
    let build_dir = format!("./build/php-{}-custom", php_version);
    tokio::fs::create_dir_all(&build_dir).await
        .map_err(|e| format!("创建构建目录失败: {}", e))?;
    
    tokio::fs::write(format!("{}/Dockerfile", build_dir), &dockerfile).await
        .map_err(|e| format!("写入 Dockerfile 失败: {}", e))?;
    
    // 5. 执行 docker build
    log::info!("🔨 开始构建镜像: {}", image_tag);
    log::info!("📦 扩展列表: {:?}", extensions);
    
    let start_time = std::time::Instant::now();
    
    let output = Command::new("docker")
        .args(&["build", "-t", &image_tag, &build_dir])
        .output()
        .await
        .map_err(|e| format!("Docker 命令执行失败: {}", e))?;
    
    let elapsed = start_time.elapsed();
    
    if !output.status.success() {
        let stderr = String::from_utf8_lossy(&output.stderr);
        
        // 分析错误类型
        if stderr.contains("timeout") || stderr.contains("network") {
            return Err(format!(
                "⚠️ 网络超时，请检查网络连接或切换镜像源\n\n错误详情:\n{}",
                stderr
            ));
        } else if stderr.contains("not found") {
            return Err(format!(
                "❌ 扩展不存在或版本不兼容\n\n错误详情:\n{}",
                stderr
            ));
        } else {
            return Err(format!(
                "❌ 镜像构建失败 (耗时: {:.2}s)\n\n错误详情:\n{}",
                elapsed.as_secs_f64(),
                stderr
            ));
        }
    }
    
    log::info!("✅ 镜像构建成功 (耗时: {:.2}s): {}", elapsed.as_secs_f64(), image_tag);
    
    // 6. 清理构建目录（可选）
    // tokio::fs::remove_dir_all(&build_dir).await.ok();
    
    Ok(image_tag)
}

/// 生成唯一的镜像标签
fn generate_image_tag(php_version: &str, extensions: &[String]) -> String {
    let mut sorted_exts = extensions.to_vec();
    sorted_exts.sort();
    
    // 使用扩展列表的哈希作为标签的一部分
    let hash = calculate_hash(&sorted_exts.join(","));
    let short_hash = &hash.to_string()[..8];
    
    format!("php:{}-custom-{}", php_version, short_hash)
}

/// 检查镜像是否存在
async fn image_exists(image_tag: &str) -> bool {
    use tokio::process::Command;
    
    let output = Command::new("docker")
        .args(&["image", "inspect", image_tag])
        .output()
        .await;
    
    output.map_or(false, |o| o.status.success())
}

/// 计算哈希值
fn calculate_hash(input: &str) -> u64 {
    use std::collections::hash_map::DefaultHasher;
    use std::hash::{Hash, Hasher};
    
    let mut hasher = DefaultHasher::new();
    input.hash(&mut hasher);
    hasher.finish()
}
```

```rust
    for (name, preset_exts) in PRESET_EXTENSIONS {
        if extensions.len() == preset_exts.len() {
            let mut sorted_user = extensions.clone();
            let mut sorted_preset: Vec<String> = preset_exts.iter().map(|s| s.to_string()).collect();
            sorted_user.sort();
            sorted_preset.sort();
            
            if sorted_user == sorted_preset {
                return Some(name);
            }
        }
    }
    None
}
```

**优化策略**：
- 如果用户选择的扩展匹配预设组合，直接使用预构建镜像（秒级启动）
- 否则，动态构建自定义镜像（5-10 分钟，但会缓存）

```rust
pub fn generate_init_sql(databases: &[DatabaseSpec]) -> String {
    let mut sql = String::new();
    
    for db in databases {
        sql.push_str(&format!(
            "CREATE DATABASE IF NOT EXISTS `{}` CHARACTER SET {} COLLATE {};\n",
            db.name, db.charset, db.collation
        ));
        
        for user in &db.users {
            sql.push_str(&format!(
                "CREATE USER IF NOT EXISTS '{}'@'%' IDENTIFIED BY '{}';\n",
                user.username, user.password
            ));
            sql.push_str(&format!(
                "GRANT {} ON `{}`.* TO '{}'@'%';\n",
                user.privileges.join(", "),
                db.name,
                user.username
            ));
        }
        
        sql.push('\n');
    }
    
    sql
}
```

```rust
/// 生成 Nginx 站点配置
pub fn generate_nginx_site_config(config: &NginxConfig) -> Result<String, String> {
    let ssl_block = if config.ssl_enabled {
        if config.ssl_cert_path.is_none() || config.ssl_key_path.is_none() {
            return Err("SSL 启用时必须提供证书和密钥路径".to_string());
        }
        
        format!(
            r#"    listen 443 ssl http2;
    ssl_certificate {};
    ssl_certificate_key {};
    ssl_protocols TLSv1.2 TLSv1.3;
    ssl_ciphers HIGH:!aNULL:!MD5;"#,
            config.ssl_cert_path.as_ref().unwrap(),
            config.ssl_key_path.as_ref().unwrap()
        )
    } else {
        "    listen 80;".to_string()
    };
    
    let php_fpm_block = if config.php_fpm_host.is_empty() {
        String::new()
    } else {
        format!(
            r#"
    location ~ \.php$ {{
        fastcgi_pass {};
        fastcgi_index index.php;
        fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
        include fastcgi_params;
    }}"#,
            config.php_fpm_host
        )
    };
    
    let additional_configs = config.additional_configs.iter()
        .map(|(key, value)| format!("    {}: {};", key, value))
        .collect::<Vec<_>>()
        .join("\n");
    
    Ok(format!(
        r#"server {{
{}
    server_name {};
    root {};
    index index.php index.html index.htm;

    access_log /var/log/nginx/access.log;
    error_log /var/log/nginx/error.log;

{}

    location / {{
        try_files $uri $uri/ /index.php?$query_string;
    }}

{}

{}
}}"#,
        ssl_block,
        config.server_name,
        config.root_path,
        php_fpm_block,
        additional_configs
    ))
}
```

```rust
fn validate_compatibility(spec: &EnvironmentSpec) -> Result<(), String> {
    // 1. 检查 PHP 版本是否存在
    // 2. 检查 MySQL/Redis 版本是否与 PHP 兼容
    // 3. 检查端口是否冲突
    // 4. 检查数据目录是否可用
    
    for service in &spec.services {
        match service.software_type {
            SoftwareType::PHP => {
                // 验证扩展兼容性
                if let Some(extensions) = &service.extensions {
                    for ext in extensions {
                        if !is_extension_available(&service.version, ext) {
                            return Err(format!("PHP {} 不支持扩展: {}", service.version, ext));
                        }
                    }
                }
            }
            _ => {}
        }
    }
    
    Ok(())
}
```

## ⚠️ 风险与应对

### 风险 1：PHP 扩展安装网络问题 🔴

**问题**：`docker-php-ext-install` 和 `pecl install` 访问 GitHub、PECL 官网时可能超时

**应对**：
- ✅ 默认配置阿里云镜像源（apt、pypi、composer）
- ✅ 提供镜像源切换选项（阿里云、清华、自定义）
- ✅ Dockerfile 中配置国内 apt 源
- ✅ 构建失败时自动重试（最多 3 次）
- ✅ 显示详细的错误日志和解决建议

### 风险 2：版本兼容性复杂 🟡

**问题**：PHP 扩展、MySQL 版本、Redis 版本之间的兼容性矩阵复杂

**应对**：
- ✅ 优先支持主流组合（现代栈、稳定栈）
- ✅ 提供“推荐组合”选项
- ✅ 允许高级用户手动选择（带警告）
- ✅ 内置兼容性检查逻辑

### 风险 3：部署失败率 🟡

**问题**：网络问题、端口冲突、镜像拉取失败等导致部署失败

**应对**：
- ✅ 部署前预检查（端口、网络、磁盘空间）
- ✅ 失败时提供清晰的错误信息
- ✅ 支持重试机制
- ✅ 自动回滚已创建的容器

### 风险 4：性能问题 🟢

**问题**：同时拉取多个镜像、构建自定义镜像导致时间长

**应对**：
- ✅ 并行拉取镜像（限制并发数）
- ✅ 显示实时进度
- ✅ 镜像缓存复用（相同扩展组合不重复构建）
- ✅ 使用国内镜像源
- ✅ 提供预设镜像标签（秒级启动）

### 风险 5：数据安全性 🟡

**问题**：数据库密码、API 密钥等敏感信息泄露

**应对**：
- ✅ 使用 `.env` 文件存储敏感信息
- ✅ `.env` 文件不提交到 Git
- ✅ 提供密码强度验证
- ✅ 支持自定义 root 密码

### 风险 6：Nginx 配置复杂性 🟢

**问题**：Nginx 配置语法错误导致服务无法启动

**应对**：
- ✅ 生成配置后自动执行 `nginx -t` 测试
- ✅ 提供常用配置模板
- ✅ 重载前备份旧配置
- ✅ 失败时自动回滚
